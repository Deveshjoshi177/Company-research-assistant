# 🤖 AI-Powered Company Research Assistant

A modern, responsive ChatGPT-style web application built for the hiring hackathon. This application allows users to research any target company by name or URL. It intelligently crawls internal pages, aggregates public web indices using Serper.dev, synthesizes deep market intelligence via OpenRouter AI models, identifies key competitors, generates downloadable PDF reports, and features automated tracking dispatch workflows to Discord channels.

## 🚀 Core Features
- **Intelligent Dual-Input Routing:** Seamless handling of both loose corporate names and explicit domain URLs.
- **Advanced Target Web Crawler:** Custom engine designed to discover internal landing pages, extract clean body text copies, and filter out authentication gateways.
- **Multi-Source Knowledge Synthesis:** Marries internal site assets with public records to assemble granular details (Contact Maps, Core Offerings).
- **Interactive LLM Interface:** A clean conversational canvas with dynamic multi-model selection arrays.
- **Corporate PDF Engine:** Instant 1-click generation of professional corporate summary dossiers.
- **Discord Bot Automation Integration (Bonus):** Instant background data-routing to evaluation target endpoints.
